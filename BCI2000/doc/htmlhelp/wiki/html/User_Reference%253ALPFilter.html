Function

The LPFilter is a simple single-pole temporal low pass filter with a time constant T (given in units of a sample's duration), a sequence Sin,t as input and a sequence Sout,t as output (where t is a sample index proportional to time), and obeying

Typically, the LPFilter would be used to remove high-frequency noise from the Classifier's output.

Parameters

LPTimeConstant

The filter's time constant in units of sample blocks, or in seconds if followed with an "s".

Consistent with its mathematical limit, a value of 
LPTimeConstant= 0
disables the LPFilter.

States

None.
