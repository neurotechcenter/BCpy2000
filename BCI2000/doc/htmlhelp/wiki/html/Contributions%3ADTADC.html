Installation

The DTADC supports multiple connected boards. To connect multiple boards, follow these instructions:

Parameters

SoftwareChBoard1

The number of channels for board 1. Ignored in case of a single board.

SoftwareChBoard2

The number of channels for board 2. Ignored in case of a single board.

BoardName

The name of the AD board.

BoardName2

The name of the second AD board, or 'none' if there is no second board present.

States

None.
